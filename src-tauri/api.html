STAC API - Core Documentation

This is the HTML documentation for the ZenSTAC API implementation. This API implements the STAC API Core specification.

Conformance

This API implements the following conformance classes:

  • STAC API - Core
  • STAC API - Collections
  • STAC API - Items
  • STAC API - Item Search
  • STAC API - Features
  • STAC API - Item Search Sortables
  • OGC API - Features - Sortables
  • STAC API - Collection Search Sortables
  • OGC API - Features - Core
  • OGC API - Features - OpenAPI 3.0
  • OGC API - Features - HTML
  • OGC API - Features - GeoJSON

Endpoints

GET
/
Returns the root STAC catalog.

GET /search

Searches for items across all collections using query parameters.

POST /search

Searches for items across all collections using a POST body.
